# Car Dealership Application

## Description
This is a Java console application that allows users to manage a car dealership inventory.

Users can:
- View all vehicles
- Search vehicles by:
  - Price
  - Make and model
  - Year
  - Color
  - Mileage
  - Vehicle type
- Add vehicles
- Remove vehicles

The application reads and saves data using a pipe-delimited CSV file.

## Technologies Used
- Java
- IntelliJ IDEA
- Maven
- Git & GitHub

## Features
- Object-oriented design
- CSV file reading/writing
- Console menu system
- Vehicle filtering and searching
